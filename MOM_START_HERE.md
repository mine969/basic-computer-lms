# Start Here

For your mother, use the simplest path:

1. Double-click `Create Desktop Shortcut.bat` one time.
2. A shortcut named `Computer Lessons` will appear on the Desktop.
3. Double-click `Computer Lessons` when she wants to study.
4. The lessons will open in the browser.
5. Keep the small black window open while studying.
6. Close the black window when finished.

Progress saves automatically on the same computer.

If the browser does not open, double-click:

```txt
Computer Lessons.bat
```

If the app says Node.js is missing, install Node.js once, then try again.
